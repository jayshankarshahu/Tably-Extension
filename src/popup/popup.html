Tably Extension

Simulate Alt+Tab behaviour for your chrome tabs

⚠️

Note

This extension may not work on certain sites like chrome:// pages or the Chrome Web Store due to Chrome's security restrictions.

⚙️

Settings

Show preview Show a preview of the tab when navigating
All windows Show tabs from all windows instead of current window only
